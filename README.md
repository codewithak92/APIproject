# Task Manager Application - Backend Developer Intern Assignment

This project is a scalable REST API built with FastAPI (Python) and a modern Vanilla JS Frontend, developed for the Backend Developer Intern assignment.

## Features
### Backend (Primary)
- **FastAPI** with Python 3
- **SQLAlchemy** ORM & **PostgreSQL** database (default configuration in `.env`)
- **JWT Authentication** and secure password hashing using bcrypt
- **Role-Based Access Control** (`user` and `admin` roles)
- **CRUD Operations** for Tasks
- Complete **Swagger API Documentation** available at `/docs`

### Frontend (Secondary)
- Built with **Vanilla JavaScript, HTML, and CSS** (No Node.js/NPM required!)
- **Dynamic SPA** behavior without page reloads
- **Modern Aesthetic**: Glassmorphism, dynamic animations, dark mode theme
- Full integration with the API (Registration, Login, CRUD Tasks)

## Quick Start (Local Development)

### 1. Backend Setup
Make sure you have Python 3 installed.
```bash
cd backend

# Create and activate a virtual environment (optional but recommended)
python -m venv venv
# Windows: venv\Scripts\activate
# Mac/Linux: source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Start the server (runs on port 8000 by default)
cd app
uvicorn app.main:app --reload
```
> **Note on Database:** The `.env` file is pre-configured with `DATABASE_URL=postgresql://postgres:#ayanayanx2@localhost:5432/taskmanager`. Ensure your local PostgreSQL is running with these credentials and the database exists, OR modify `.env` to point to an SQLite database (e.g., `sqlite:///./test.db`).

### 2. Frontend Setup
Since the frontend uses pure Vanilla HTML/CSS/JS, no build step is needed.
- Open `frontend/index.html` directly in your web browser.
- Alternatively, use a simple HTTP server to serve the directory:
  ```bash
  cd frontend
  python -m http.server 8080
  ```
  Then visit `http://localhost:8080` in your browser.
